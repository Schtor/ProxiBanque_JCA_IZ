	************************************
*****  	Description de l'application:   	*****
	************************************

ProxiBanqueSI est une application permettant à un conseiller de gérer ses clients et les demandes de ses derniers


	************************************
*****  		Fonctionnalités:		*****
	************************************

L'application permet de :
- se connecter en tant que conseiller
(- consulter la liste des clients
- editer les informations des clients
- consulter les comptes des clients
- effectuer des virements entre clients) En cours de développement

Le nom de la base de données est ProxiBanque_IZ_JCA
	
	************************************
*****  		Livrables:			*****
	************************************
	
Vous trouverez les elements suivants:

1) la documentation au format javadoc : "/doc".

2) les sources du projet : dans le dossier "/src".

3) la conception (Diagrammes de classe et de déploiement ainsi que la cinématique des pages web).

4) le lien git contenu dans le fichier git.txt.

5) le fichier readme.txt : ce document.


	************************************
*****  	Documentation:			  	*****
	************************************

Veuillez suivre les indications suivantes:

1) Rendez vous dans le dossier 'doc/'.

2) Double cliquez sur 'index.html'.

	
	************************************
***** 	Execution de l'application:    		*****
	************************************

Veuillez suivre les indications suivantes:

1) Copier le dossier Proxibanque_IZ_JCA dans votre workspace
	
2) Importer l'intégralité du dossier du projet dans votre IDE

3) Exécuter le projet sur un serveur local type Tomcat
